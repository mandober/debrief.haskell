# Haskell2010

Haskell2010 compiles Haskell 2010 language variant.

`Haskell2010` enables the following language 12 extensions:
- CUSKs
- DatatypeContexts
- DoAndIfThenElse
- EmptyDataDecls
- FieldSelectors
- ForeignFunctionInterface
- ImplicitPrelude
- MonomorphismRestriction
- PatternGuards
- RelaxedPolyRec
- StarIsType
- TraditionalRecordSyntax
