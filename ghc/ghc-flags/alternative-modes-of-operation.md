# Alternative modes of operation

https://downloads.haskell.org/~ghc/latest/docs/html/users_guide/flags.html#alternative-modes-of-operation

```bash
--frontend ⟨module⟩
--help, -?
--info
--interactive
--make
--mk-dll
--numeric-version
--print-libdir
--show-iface ⟨file⟩
--show-options
--supported-extensions, --supported-languages
--version, -V
-e ⟨expr⟩
-M
```
