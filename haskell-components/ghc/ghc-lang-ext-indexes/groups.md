# Extension groups


## Haskell2010

The Haskell 2010 language variant enables:

+ ImplicitPrelude
+ StarIsType
+ CUSKs
+ MonomorphismRestriction
+ DatatypeContexts
+ TraditionalRecordSyntax
EmptyDataDecls
ForeignFunctionInterface
PatternGuards
DoAndIfThenElse
RelaxedPolyRec


## Haskell98

The Haskell 98 language variant enables:

+ ImplicitPrelude
+ StarIsType
+ CUSKs
+ MonomorphismRestriction
NPlusKPatterns
+ DatatypeContexts
+ TraditionalRecordSyntax
NondecreasingIndentation
